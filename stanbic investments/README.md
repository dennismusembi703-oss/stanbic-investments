# Stanbic Investments Website

A small static website for Stanbic Investments, built with HTML, CSS, and JavaScript.

## Project files

- `index.html` — main landing page
- `admin.html` — admin dashboard page
- `dashboard.html` — dashboard overview page
- `login.html` — user login page
- `admin-login.html` — admin login page
- `css/style.css` — project styles
- `js/script.js` — shared JavaScript
- `js/js/*.js` — page-specific JavaScript logic

## How to use

1. Open `index.html` in a browser.
2. Use the login pages to navigate between site views.
